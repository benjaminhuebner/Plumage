Sessions & Usage

Usage pill

The status bar can show a compact usage pill next to the session indicator; click it for the full breakdown. Configure it in Settings > Usage: the pill itself, the 5-hour window and the 7-day window toggle independently. By default the pill and the 5-hour window are on; the 7-day window is off.

Keeping the Mac awake

"Keep Mac awake during sessions" in Settings > General (off by default) prevents system idle sleep while any Claude session is live. That covers the embedded session as well as sessions running in terminal tabs. Only system sleep is held off; the display may still sleep. Once no session is live, normal sleep behavior returns.
